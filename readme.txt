Ограничения Freemarker-плагина требуют, чтобы Common.kt.ftl находился в папке шаблона,
поэтому Common.kt.ftl раскопированы по всем шаблонам.
Наиболее новая обновлённая версия Common.kt.ftl находится в этой папке и считается "главной".
Но в работе самих шаблонов этот Common не используется (используется только как источник для копипаста).
